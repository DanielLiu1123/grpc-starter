介绍
----

该项目主要为 gRPC 生态提供一系列 Spring Boot starters，提供自动配置和高度可扩展的能力，让 Spring Boot 和 gRPC 可以更好地结合。

## 核心功能

* 自动装配 gRPC server

* 自动装配 gRPC client

  - 没有引入额外注解，可以直接使用 `@Autowired`，`@Resource` 等注解注入 gRPC client，完全遵从 Spring Bean 的生命周期。

* 高度可扩展
